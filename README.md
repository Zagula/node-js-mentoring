# NODE-JS-mentoring

## **Hometasks:**
#### 1. NPM/Modules
```
npm run hw1
```
#### 2. Nodemon/Watcher
```
npm run hw2
```
#### 3. CommandLine/Streams
```
npm run hw3
npm run streams -- [...args]
```
#### 4. Middleware/Frameworks
```
npm run hw4-plain
npm run hw4-html -- --sync
npm run hw4-html
npm run hw4-json
npm run hw4-echo
npm run hw4 or npm start
```

*\***current** hometask can also be executing by `npm start`*
